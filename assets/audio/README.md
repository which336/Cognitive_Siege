# Cognitive Siege Audio

These WAV files are generated by `npm run audio:generate`.

The generator lives in `scripts/generate-audio.mjs` and uses only Node.js built-in
APIs to render a cohesive "dream tower defense" palette:

- soft chimes for UI / build actions
- short airy attacks for tower fire
- darker pulses for enemy damage and SAN hits
- longer harmonic stingers for victory / game over

No external binary audio source is required. The files can be replaced later
with CC0 packs such as Kenney Interface Sounds / UI Audio if a different sonic
identity is desired.
